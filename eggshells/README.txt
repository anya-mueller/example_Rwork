This folder contains the scripts, outputs, and dummy data for the eggshell project. This project is ongoing. It is examining the variation in eggshell bacteria communities of two wild passerines. One species was sampled at two sites. This project has two components thus far:

Folder name - folder purpose

field - this folder contains the scripts, outputs, and dummy data for the field component of the project
microbiome - this folder contains the scripts, outputs, and dummy data for the metagenomic pilot project
